Plantilla backend - Municipalidad Yau
-----------------------------------
Instrucciones rápidas:

1) Copia .env.example a .env y ajusta valores.
2) Instala dependencias: npm install
3) Crea la base de datos en PostgreSQL y el usuario.
4) Ejecuta migraciones: npm run migrar
5) Levanta en desarrollo: npm run dev
6) Documentación: http://localhost:4000/api/docs (si instalas swagger)

Archivos principales:
- src/servidor.js
- src/aplicacion.js
- src/configuracion/
- migraciones/
